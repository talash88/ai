# AI Image Generator 🎨

A creative AI image generation interface that allows users to create visuals by entering prompts and selecting different options.

## ✨ Features

- Modern and responsive AI image generator UI.
- Custom prompt input section.
- Multiple AI model selection options.
- Image count and aspect ratio controls.
- Gradient-based modern design.
- Smooth hover effects and transitions.
- Font Awesome icons integration.
- Responsive layout using Flexbox and CSS Grid.

## 🔧 Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts
